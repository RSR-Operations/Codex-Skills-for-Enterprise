# Decision Memo Patterns

## One-Page Memo

Use when the decision needs a fast executive read.

```text
Decision Required
Recommendation
Why Now
Options
Risks
Next Steps
```

## Weighted Matrix

Use when multiple options are plausible and the user asks for a rigorous comparison.

| Criterion | Weight | Option A | Option B | Option C |
| --- | ---: | ---: | ---: | ---: |
| Business impact | 30% | ... | ... | ... |
| Speed | 20% | ... | ... | ... |
| Risk | 20% | ... | ... | ... |
| Cost | 15% | ... | ... | ... |
| Reversibility | 15% | ... | ... | ... |

## Executive Committee Memo

Emphasize enterprise impact, governance, risk, dependencies, and the specific approval requested.
