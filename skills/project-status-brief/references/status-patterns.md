# Project Status Brief Patterns

## RAG Health Guidance

- **Green**: on track; no material leadership action needed.
- **Yellow**: manageable risk; timeline, scope, quality, or dependency requires attention.
- **Red**: material risk to outcome; leadership intervention likely required.
- **Unknown**: source material is insufficient to judge health.

## Dashboard Format

| Field | Value |
| --- | --- |
| Status | Green / Yellow / Red / Unknown |
| Owner | ... |
| Current milestone | ... |
| Next milestone | ... |
| Top risk | ... |
| Leadership ask | ... |

## Launch Readiness

Assess readiness across product, operations, support, legal/compliance, communications, and customer impact.
